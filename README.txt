Comment cloner un fichier git ?
Aller sur le repo: cliquer sur clone or download. 
récupérer le lien et sur ligne de cmd se positionner dans le bon dossier et faire:
git clone + adresse du clone téléchargé

Pour initier un projet :
git init
Repertorie depots distants actuellement configuré: 
git remote –v

se connecter a un serveur distant: 
git remote add origin <93.188.160.58>
